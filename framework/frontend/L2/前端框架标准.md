# 前端框架模块标准（L2）
<!--@spec id=frontend.module.standard; module=frontend; level=L2; version=6-->
<!--@layer module=frontend; level=L2-->

## 1. 目标（Goal）

建立前端模块中枢能力层，承接令牌层并向组件层提供稳定服务抽象。

<!--@goal id=G1_FRONTEND_L2-->

## 2. 边界定义（Boundary）

- `SERVICE_SCOPE`：通用服务范围（权限、会话、消息、国际化）。  <!--@boundary symbol=SS; name=service_scope; type=ServiceScope; trace=BoundaryDefinition.service_scope-->
- `GUARD_SCOPE`：守卫策略范围（路由、身份、特性开关）。  <!--@boundary symbol=GS; name=guard_scope; type=GuardScope; trace=BoundaryDefinition.guard_scope-->
- `CONFIG_SCOPE`：配置治理范围（配置分层、覆盖策略）。  <!--@boundary symbol=CS; name=config_scope; type=ConfigScope; trace=BoundaryDefinition.config_scope-->

<!--@boundary_set id=BD_FRONTEND_L2-->

## 4. 组合原则（Combination Principles）

- 同层禁止组合：`L2 -> L2` 非法。
- 反向组合禁止：`L3 <- L2` 非法。
- 跨层跳级禁止：`L2 -> L4` 非法。
- 仅允许下游向上游组合：`L2 -> L3`。

组合约束集合：
- `C1` 相邻层约束
- `C2` 下游上行约束
- `C3` 服务到组件能力契约

组合映射（示例）：
- `U1 = {L2.B1, L2.B2, L2.B6} --[C1,C2,C3]--> L3.B1`
- `U2 = {L2.B1, L2.B3} --[C1,C2,C3]--> L3.B2`
- `U3 = {L2.B4, L2.B5} --[C1,C2,C3]--> L3.B4`

- 通用服务基组合到复合组件基。  <!--@compose from=L2.Mfrontend.B1; to=L3.Mfrontend.B2; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 通用服务基组合到组件原子基。  <!--@compose from=L2.Mfrontend.B1; to=L3.Mfrontend.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 路由守卫基组合到组件原子基。  <!--@compose from=L2.Mfrontend.B2; to=L3.Mfrontend.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 配置治理基组合到表单域组件基。  <!--@compose from=L2.Mfrontend.B3; to=L3.Mfrontend.B3; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 配置治理基组合到复合组件基。  <!--@compose from=L2.Mfrontend.B3; to=L3.Mfrontend.B2; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 国际化基组合到反馈组件基。  <!--@compose from=L2.Mfrontend.B4; to=L3.Mfrontend.B4; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 消息中心基组合到可访问适配基。  <!--@compose from=L2.Mfrontend.B5; to=L3.Mfrontend.B5; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 消息中心基组合到表单域组件基。  <!--@compose from=L2.Mfrontend.B5; to=L3.Mfrontend.B3; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 会话缓存基组合到组件原子基。  <!--@compose from=L2.Mfrontend.B6; to=L3.Mfrontend.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->

## 5. 验证（Verification）

1. 层级编号必须由目录 `Lx/` 决定。  <!--@verify_item layer=structure; check=level_from_directory_only-->
2. 模块节点必须可解析为 `L{X}.M{module}.B{n}`。  <!--@verify_item layer=structure; check=canonical_node_id-->
3. 组合边仅允许 `Lx -> L(x+1)`。  <!--@verify_item layer=combination; check=downstream_to_upstream_only-->
4. 下游节点允许多父引用，但必须至少有一个上游父节点。  <!--@verify_item layer=combination; check=dag_multi_parent_allowed_with_min_parent-->
