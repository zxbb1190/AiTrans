# 前端框架模块标准（L7）体验目标与验证层
<!--@layer module=frontend; level=L7-->

## 1. 目标（Goal）

定义顶层体验目标与验收门禁，作为前端模块分层结构的上游收敛根。

## 2. 边界定义（Boundary）

- `UX_SCOPE`：体验指标范围（首屏、交互、稳定性、可访问性）。
- `QA_SCOPE`：验证策略覆盖范围。
- `RELEASE_SCOPE`：发布准入与回滚门禁范围。

## 3. 最小可行基（Bases）

- `B1` 体验目标根基  <!--@base id=B1; name=UX_ROOT_BASE; capability=experience_target_root-->

## 4. 组合原则（Combination Principles）

`L7` 为该模块上游根层，仅接收来自 `L6` 的组合，不向更高层输出。

## 5. 验证（Verification）

- `L7` 节点数量应少于下游层（树顶收敛）。
- 所有进入 `L7` 的边必须来自 `L6`。
- 验证证据必须能回溯到下游链路。
