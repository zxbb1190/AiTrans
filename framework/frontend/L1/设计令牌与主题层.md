# 前端框架模块标准（L1）设计令牌与主题层
<!--@layer module=frontend; level=L1-->

## 1. 目标（Goal）

将底座能力抽象为统一视觉语义，形成可治理的主题令牌体系。

## 2. 边界定义（Boundary）

- `TOKEN_SCOPE`：令牌覆盖范围（颜色/排版/间距/动效）。
- `THEME_SCOPE`：主题数量与切换策略。
- `A11Y_TOKEN_RULE`：可访问令牌约束。

## 3. 最小可行基（Bases）

- `B1` 语义令牌基  <!--@base id=B1; name=SEMANTIC_TOKEN_BASE; capability=semantic_token_definition-->
- `B2` 主题编排基  <!--@base id=B2; name=THEME_ORCH_BASE; capability=theme_orchestration_switch-->
- `B3` 语义色彩基  <!--@base id=B3; name=COLOR_TOKEN_BASE; capability=color_token_governance-->
- `B4` 语义排版基  <!--@base id=B4; name=TYPO_TOKEN_BASE; capability=typography_token_governance-->
- `B5` 动效令牌基  <!--@base id=B5; name=MOTION_TOKEN_BASE; capability=motion_token_governance-->
- `B6` 样式审计基  <!--@base id=B6; name=STYLE_AUDIT_BASE; capability=style_lint_and_audit-->
- `B7` 可访问令牌基  <!--@base id=B7; name=A11Y_TOKEN_BASE; capability=a11y_token_constraints-->

## 4. 组合原则（Combination Principles）

仅允许下游向上游组合：`L1 -> L2`。禁止同层组合与反向组合。

组合约束集合：
- `C1` 相邻层约束
- `C2` 方向约束（下游到上游）
- `C3` 能力契约约束（token->service）

组合映射（示例）：
- `U1 = {L1.B1, L1.B2, L1.B7} --[C1,C2,C3]--> L2.B1`
- `U2 = {L1.B1, L1.B3} --[C1,C2,C3]--> L2.B2`
- `U3 = {L1.B4, L1.B5} --[C1,C2,C3]--> L2.B3`

- 语义令牌基组合到路由守卫基。  <!--@compose from=L1.Mfrontend.B1; to=L2.Mfrontend.B2; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 语义令牌基组合到通用服务基。  <!--@compose from=L1.Mfrontend.B1; to=L2.Mfrontend.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 主题编排基组合到通用服务基。  <!--@compose from=L1.Mfrontend.B2; to=L2.Mfrontend.B1; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 语义色彩基组合到配置治理基。  <!--@compose from=L1.Mfrontend.B3; to=L2.Mfrontend.B3; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 语义色彩基组合到路由守卫基。  <!--@compose from=L1.Mfrontend.B3; to=L2.Mfrontend.B2; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 语义排版基组合到国际化基。  <!--@compose from=L1.Mfrontend.B4; to=L2.Mfrontend.B4; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 语义排版基组合到配置治理基。  <!--@compose from=L1.Mfrontend.B4; to=L2.Mfrontend.B3; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 动效令牌基组合到消息中心基。  <!--@compose from=L1.Mfrontend.B5; to=L2.Mfrontend.B5; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 样式审计基组合到会话缓存基。  <!--@compose from=L1.Mfrontend.B6; to=L2.Mfrontend.B6; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->
- 可访问令牌基组合到国际化基。  <!--@compose from=L1.Mfrontend.B7; to=L2.Mfrontend.B4; rule=adjacent_only; principle=P_DOWNSTREAM_UPSTREAM; constraint=C1_C2_C3-->

## 5. 验证（Verification）

- 每个 L1 基必须至少指向一个 L2 父节点。
- 每个 L2 基必须至少由一个 L1 子节点支撑。
- 组合方向必须为 `L1 -> L2`。
