# 窗帘框架标准（L2）
<!--@spec id=curtain.L2.standard; level=L2; domain=curtain; version=2-->
<!--@layer module=curtain; level=L2-->

## 1. 目标（Goal）

定义可安装、可调节的遮光装置，以提升空间光照控制效率与隐私保障能力。

## 2. 边界定义（Boundary）

- `W`：适配窗宽
- `H`：适配窗高
- `L`：遮光等级
- `D`：驱动方式（手动/电动）
- `S`：安全约束（防夹、防坠）

## 3. 最小可行基（Bases）

- `B1` 幕布组件  <!--@base id=B1; name=FABRIC; capability=light_block_surface-->
- `B2` 导轨组件  <!--@base id=B2; name=TRACK; capability=open_close_path-->
- `B3` 驱动控制组件  <!--@base id=B3; name=DRIVE; capability=drive_and_safety_stop-->

## 4. 组合原则（Combination Principles）

- `R1`：导轨与幕布必须形成完整开合路径。
- `R2`：电动驱动方案必须包含安全停止机制。

## 5. 验证（Verification）

通过条件：
- 安装适配参数有效。
- 组合满足开合与安全规则。
- 目标控制效果优于基线方案。
