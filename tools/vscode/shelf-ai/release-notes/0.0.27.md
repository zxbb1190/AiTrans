# Shelf AI 0.0.27

## 中文说明

- 这一版把 Shelf AI 的整体界面继续往更现代的 VS Code 插件风格收紧：侧边栏主页重做成更紧凑的 home view，突出状态总览、快速操作、工作区信号和问题预览，而不是单纯堆叠大卡片。
- 框架树页也做了结构级优化：图面板现在保持原尺寸滚动，右侧统计和节点详情改成更像 inspect 面板的分组结构，整体更接近 GitLens / Git Graph 这类插件的阅读方式。
- 为了解决大图看不清的问题，这版新增了缩放能力：支持 `+`、`-`、`100%`、`适配` 按钮，以及 `Ctrl/⌘ + 滚轮` 缩放。

## English Notes

- This release further modernizes the Shelf AI UI toward a cleaner VS Code-native experience. The sidebar home view has been reworked into a denser layout centered on overview, quick actions, workspace signals, and issue preview instead of large stacked cards.
- The framework tree page also received a structural redesign: the graph now stays on a full-size scrollable surface, while the right-side statistics and node details are grouped more like an inspect panel, closer to the reading model used by extensions such as GitLens and Git Graph.
- To address readability on large graphs, this release adds graph zoom support with `+`, `-`, `100%`, `Fit`, and `Ctrl/⌘ + mouse wheel` zooming.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 安装后如未立即生效，执行一次 `Developer: Reload Window`
