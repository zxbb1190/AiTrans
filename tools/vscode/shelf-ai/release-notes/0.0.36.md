# Shelf AI 0.0.36

## 中文说明

- 这一版主要整理 Shelf AI 的公开发布入口和打包元数据，让仓库地址、问题反馈地址和主页继续与当前公开 GitHub 仓库保持一致。
- 同时补充了更明确的 WSL 安装说明。现在 README 会直接说明：如果你是在 WSL 里工作，应在 WSL 环境中执行本地安装脚本，并让 VS Code 连接到同一个 WSL 工作区。
- 这不是一版功能侧的大更新，而是一版面向发布、安装和公开入口一致性的整理版。升级后，公开链接、安装说明和当前仓库状态会保持一致。

## English Notes

- This release refreshes Shelf AI public distribution metadata and entry links so repository, issue tracker, and homepage targets stay aligned with the current public GitHub location.
- It also adds explicit WSL-aware installation guidance. The packaged README now clarifies that local installation should run inside WSL when the workspace is opened through VS Code Remote - WSL.
- This is not a feature-heavy release. It is a packaging, install-flow, and public-entry consistency release so the extension metadata matches the current repository state.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你使用 WSL，请在对应的 WSL 工作区窗口里安装并重载当前窗口
