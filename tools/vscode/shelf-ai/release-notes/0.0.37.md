# Shelf AI 0.0.37

## 中文说明

- 这一版完成了 VS Code 扩展的正式统一命名：旧的公开名称、命令前缀、媒体资源命名和发布工作流入口，现已统一为 Shelf AI，简称 Shelf。
- 本地安装脚本、扩展 README、发布工作流和版本说明约定也同步切换到新的 `shelf-ai-v*` tag 与 `shelf-ai-<version>.vsix` 安装包命名，减少仓库内外的品牌分裂。
- 同时更新了知识库示例产品的品牌文案，使示例应用与扩展、README 和仓库定位保持一致。

## English Notes

- This release completes the public rebrand of the VS Code companion to Shelf AI. Commands, packaged metadata, media asset names, and release automation now use Shelf AI consistently, with Shelf as the short form.
- The local install flow, packaged README, release workflow, and versioning convention now follow the new `shelf-ai-v*` tag and `shelf-ai-<version>.vsix` artifact naming.
- The reference knowledge-base product branding was also updated so the demo app, extension, and repository positioning now share one public name.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你使用 WSL，请在对应的 WSL 工作区窗口里安装并重载当前窗口
