# Shelf AI 0.0.28

## 中文说明

- 这一版把边界导航从“只有实例显式暴露的大类 section 才能跳”扩展成“所有有效边界都可追踪”。像 `CITATION`、`TURN`、`INPUT`、`SCOPE`、`STATUS` 这类原子边界，现在也会跳到它们在 `instance.toml` 中的主归属 section，而不再只停留在本地定义。
- 边界 hover 说明也一起增强了：现在会区分“主归属 section”和“相关 section”，并在映射是推导归属而不是直接一对一声明时，给出归属说明，方便理解为什么会跳到那个配置位置。
- 同时补上了框架 Markdown 的引用结果支持。现在对边界符号执行 `Find All References` / `Shift+F12`，不再只看到空结果；会同时返回当前使用位置、框架定义位置以及映射到的实例配置位置。

## English Notes

- This release expands boundary navigation from explicit top-level instance sections to full boundary traceability. Derived atomic boundaries such as `CITATION`, `TURN`, `INPUT`, `SCOPE`, and `STATUS` now jump to their owning `instance.toml` sections instead of stopping at the local framework definition only.
- Boundary hover cards are also richer now: they distinguish the primary owning section from additional related sections, and include an ownership note when the mapping is inferred rather than declared as a direct one-to-one section.
- This release also adds framework-markdown reference results for navigable boundary symbols. `Find All References` / `Shift+F12` now returns the current usage, the framework definition, and the mapped instance configuration target instead of empty results.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 安装后如未立即生效，执行一次 `Developer: Reload Window`
