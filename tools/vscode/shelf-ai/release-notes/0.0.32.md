# Shelf AI 0.0.32

## 中文说明

- 这一版修复了 framework 树图里“画布拖动”和“节点/连线选中”之间互相抢事件的问题。现在左键拖动背景仍然可以平移整张图，但点击模块或连线时，关系高亮、详情面板查看和连接关系检查不会再被拖动行为吞掉。
- `Ctrl/⌘ + 点击` 节点或连线的来源跳转能力继续保留，同时 framework 分组头部的拖动与折叠控件也被明确隔离，不会再误伤模块选中。
- 同时把默认关系线和箭头稍微加重了一点，让大图里边的方向更容易看清，但不会把整体画面压得太重。

## English Notes

- This release fixes the event conflict between canvas panning and node / edge selection in the framework tree. Left-dragging the background still pans the full graph, while clicking a module or relation now reliably preserves relationship highlighting, inspect-panel detail viewing, and connection inspection.
- `Ctrl/⌘ + click` source jumps on nodes and edges remain intact, and the framework-group header drag / collapse controls are now explicitly isolated so they no longer interfere with module selection.
- It also makes default relationship lines and arrowheads slightly stronger, improving direction readability on large graphs without making the overall view visually heavy.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你已经打开过框架树页面，更新后请执行一次 `Shelf: Refresh Framework Tree`，或关闭并重新打开 `Shelf AI : Framework Tree`
