# Shelf AI 0.0.24

## 中文说明

- 本次发布补齐了 Shelf AI 的公开发布说明流程：GitHub Release 不再只显示自动生成的 compare 链接，而是强制使用仓库内维护的正式版本说明。
- 发布流程现在会强制校验双语 release notes；若缺少中文说明、英文说明或 `.vsix` 安装包，工作流会直接失败。
- 当前安装包延续了最近一轮导航改进：模块引用整体可点击，点击直接跳到目标模块的首个 `B*`；悬浮可显示能力声明、最小可行基，以及组合原则中的参与基、组合方式、输出能力、边界绑定。

## English Notes

- This release fixes the public release-note workflow for Shelf: GitHub Releases no longer rely on auto-generated compare text only, and now require a repository-backed release-note document.
- The publish pipeline now validates bilingual release notes; the workflow fails if the Chinese section, English section, or the `.vsix` asset is missing.
- The shipped package continues to include the recent navigation improvements: module references are treated as one clickable target, jump directly to the target module's first `B*`, and show hover summaries for capabilities, bases, and rule details.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 安装后如未立即生效，执行一次 `Developer: Reload Window`
