# Shelf AI 0.0.31

## 中文说明

- 这一版把 framework 树图进一步改成“可操作的 framework 分组视图”。每个 `framework/<name>` 组现在都带有自己的头部控制区，可以单独折叠或展开，并且能够作为一个整体在图里拖动位置。
- 新增了 `恢复布局` 按钮。手动拖动过各个 framework 组之后，可以一键回到默认的紧凑布局；默认位置仍然按跨 framework 引用关系自动排布。
- 同时修复了 framework 组拖动命中区过于脆弱的问题。现在拖动不再依赖容易失效的透明标题区域，而是使用稳定的组级拖动命中区，并保持节点 hover、点击和 `Ctrl/⌘ + 点击` 跳转能力不回归。

## English Notes

- This release upgrades the framework tree into a more interactive framework-group view. Each `framework/<name>` panel now has its own header controls, can be collapsed or expanded independently, and can be dragged as a whole inside the graph.
- A new `恢复布局` action restores the default compact layout after manual panel rearrangement. The default positions still follow the cross-framework reference graph.
- It also fixes fragile framework-panel dragging hit zones. Dragging now uses a stable group-level handle instead of relying on easy-to-miss transparent title areas, while node hover, click, and `Ctrl/⌘ + click` jump behavior remain intact.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 如果你已经打开过框架树页面，更新后请执行一次 `Shelf: Refresh Framework Tree`，或关闭并重新打开 `Shelf AI : Framework Tree`
