# Shelf AI 0.0.34

## 中文说明

- 这一版是一次包装与说明同步发布，目的是让 Shelf AI 的版本元数据跟上仓库里已经落地的两类变化：一是 framework tree 的布局、箭头和拖动画布交互收紧；二是 `product_spec.toml` / `implementation_config.toml` 现在默认采用更详细的中文注释来承载产品讨论和实现细化。
- 这次发布不新增扩展侧命令，也不修改 Shelf AI 的命令入口。主要用户可见变化来自仓库侧生成器、生成产物和规范文档的更新，因此更准确地说，这是一次“跟随仓库当前工作流”的版本同步，而不是新的插件功能版。
- 如果你想看到更紧凑的框架树排布、更清楚的箭头呈现和更稳定的拖动画布行为，请在升级后对当前仓库执行一次 `Shelf: Refresh Framework Tree`，让工作区重新生成最新 HTML / JSON 产物。

## English Notes

- This is a packaging and release-notes sync release. Its purpose is to keep Shelf AI's published version metadata aligned with two repository-side changes that are already in effect: tighter framework-tree layout / arrow / pan behavior, and the new detailed Chinese-first authoring convention for `product_spec.toml` / `implementation_config.toml`.
- No new extension-side commands are introduced in this release. The main visible differences come from refreshed repository generators, generated artifacts, and specification-writing guidance, so this is best understood as a release that tracks the current repository workflow rather than a new plugin feature drop.
- If you want the tighter framework-tree spacing, clearer arrows, and more stable canvas drag behavior, run `Shelf: Refresh Framework Tree` after upgrading so the workspace regenerates the latest HTML / JSON artifacts.

## 安装 / Install

- 从本页面下载对应版本的 VSIX 安装包
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 升级后若框架树页面仍显示旧布局，请执行一次 `Shelf: Refresh Framework Tree`，或关闭并重新打开 `Shelf AI : Framework Tree`
