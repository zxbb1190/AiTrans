# ArchSync 0.0.29

## 中文说明

- 这一版修复了框架树交互回归问题：节点悬浮提示在命中区附近不再闪烁，普通点击恢复为稳定选中节点，`Ctrl/⌘ + 点击` 也重新可以直接跳到对应文档。
- 右侧“层级统计 / 关系统计 / 节点详情”侧栏的显隐也修好了。现在工具条里的“隐藏侧栏”会真正收起侧栏，再次点击可以恢复显示，不会再出现按钮变了但布局没收起的情况。
- 这次发布的主要修复点在仓库侧的框架树生成模板和生成产物交互逻辑，因此更新后应重新执行一次框架树刷新或重新生成 `docs/hierarchy/shelf_framework_tree.html`，以拿到修复后的页面行为。

## English Notes

- This release fixes framework-tree interaction regressions in the generated graph page: node hover no longer flickers near the hit zone, normal click selection works again, and `Ctrl/⌘ + click` jump-to-document is restored.
- The right-side stats / details panel toggle is also fixed. The toolbar's hide/show action now truly collapses and restores the side panel instead of changing the button state while leaving the layout visually open.
- The main fix in this release is in the repository-side framework-tree generator and generated HTML interaction logic, so after updating you should refresh or regenerate `docs/hierarchy/shelf_framework_tree.html` to pick up the corrected behavior.

## 安装 / Install

- 从本页面下载 `archsync-0.0.29.vsix`
- 在 VS Code 中执行 `Extensions: Install from VSIX...`
- 若你使用仓库内的框架树页面，请更新后执行一次 `ArchSync: Refresh Framework Tree`
