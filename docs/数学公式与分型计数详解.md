# 数学公式与分型计数详解（双族）

本文件与 [type_count_explained.md](/home/xue/code/shelf/docs/type_count_explained.md) 保持一致，作为中文命名入口。

## 1. 双族定义
- `Omega_frame`：FRAME（`ROD + CONNECTOR`，`panels=empty`）
- `Omega_shelf`：SHELF（`ROD + CONNECTOR + PANEL`，`panels!=empty`）
- `Omega = Omega_frame ⊔ Omega_shelf`

## 2. 规则适用域
- `R3`：全结构适用。
- `R4/R5`：仅 SHELF 适用。
- FRAME 下 R4/R5 视为 `not applicable / pass`，使用 frame-specific 规则：
  - `connected`
  - `ground_contact`
  - `minimal_under_deletability`
  - 可选 `forbid_dangling_rods`

## 3. 计数公式
### 3.1 SHELF
- `T_non_empty = sum_{Q subseteq G, Q!=empty} tau(Q)`
- `T_layer = delta_empty + T_non_empty`
- `C_raw^shelf = (T_layer)^N`

### 3.2 FRAME（V1 腔体诱导骨架）
- `C = {0..m-1}x{0..n-1}x{0..N-1}`
- 枚举 `U subseteq C, U!=empty, connected_6(U)`
- `S_frame(U)=(V_boundary(U), E_boundary(U), empty)`
- `C_raw^frame = |{U subseteq C | U!=empty and connected_6(U)}|`

### 3.3 总计数
- `C_raw^all = C_raw^frame + C_raw^shelf`
- `C_type^all = C_type^frame + C_type^shelf`
- `C_valid^all = C_valid^frame + C_valid^shelf`

## 4. Canonical 约束
- key 包含 `family`
- `xy` 平面 D4 对称
- 禁止 `z <-> x/y` 全置换
- 保留平移归一

## 5. V1 边界
FRAME 当前不是“任意杆图全覆盖”，而是“腔体诱导骨架”子空间。这是置物架语义下更稳定、可解释、可枚举的 V1 基线。
